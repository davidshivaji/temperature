# temperature

## Description
Basic weather table reading from csv.

## Setup
```
git clone git@github.com:davidshivaji/temperature.git
cd temperature
```
```
cd backend
npm i
npm start
```
To run the frontend in a separate terminal:
```
cd frontend
npm i
npm start
```
Load [](http://localhost:3000/)http://localhost:3000/ in a browser.
